# Arduino Öntözőrendszer
## Bemutatkozás
Ebben a projektben egy arduino alapú automata öntözőrendszer kerül bemutatásra. Ez a rendszer egy Arduino Uno segítségével egy kapacitív talajnedvesség érzékelő által méri a termőföld nedvességszázalékát és alapján működtet egy szivattyút, amely 3 másodpercig működik, ha a föld nedvességszázaléka egy adott érték alá esik. Továbbá integrálásra került egy ultrahangos távolságérzékelő, amely arra szolgál, hogy méri a tartályban található vízmennyiséget, így a pumpa sosem fog szárazan menni.
